# AI Pizza Ordering Agent 🍕

Голосовий ШІ-асистент для прийому замовлень на піцу в реальному часі. Побудований на базі **LiveKit Agents** та **OpenAI Realtime API**, цей агент природно спілкується з клієнтами, презентує меню, перевіряє наявність страв та оформлює замовлення.

## Технології (Tech Stack)
- **LiveKit Agents**: Для потокового WebRTC-аудіо та керування сесіями агента в реальному часі.
- **OpenAI Realtime API (`gpt-realtime-mini`)**: Для надшвидкої генерації відповідей (voice-to-voice) із мінімальною затримкою.
- **Python**: Основна логіка агента та реалізація інструментів (tools).

## Можливості (Features)
- 🎙️ **Голосова взаємодія в реальному часі**: Миттєві аудіовідповіді та живе спілкування без затримок.
- 📋 **Динамічне підключення меню**: Використовує виклики функцій (function calling) для перевірки меню, цін та наявності інгредієнтів.
- 🛒 **Керування замовленнями**: Збирає деталі замовлення (позиції, кількість, доставка/самовивіз) та підтверджує їх з клієнтом.

## Встановлення та запуск (Setup)
1. Клонуйте репозиторій.
2. Встановіть залежності:
   ```bash
   pip install -r requirements.txt
   ```
3. Створіть файл `.env` та вкажіть ключі від LiveKit та OpenAI:
   ```env
   LIVEKIT_URL=wss://...
   LIVEKIT_API_KEY=...
   LIVEKIT_API_SECRET=...
   OPENAI_API_KEY=...
   ```
4. Запустіть агента:
   ```bash
   python agent.py dev
   ```
